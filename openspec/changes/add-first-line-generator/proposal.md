## Why

Начать импровизационную сцену с чистого листа трудно: игроки тратят время на
поиск первой фразы вместо игры. Готовая яркая первая реплика мгновенно задаёт
ситуацию, отношения и напряжение, оставляя развитие сцены игрокам. Новый
генератор дополнит каталог самостоятельным инструментом с большой подборкой
открывающих реплик.

## What Changes

- Добавляется отдельное приложение «Первая реплика», выдающее готовую
  открывающую фразу для старта сцены.
- Приложение содержит не менее 200 уникальных первых реплик на русском языке.
- Каждой реплике присвоен один тон-тег из небольшого фиксированного набора
  (8 тонов), который показывается вместе с репликой.
- Пользователь может фильтровать выдачу по одному тону или получать реплики из
  всех тонов.
- Результаты перемешиваются и не повторяются до исчерпания текущей колоды
  (полной подборки или подборки выбранного тона).
- На экране показывается прогресс — сколько реплик уже показано из общего числа
  в текущей колоде.
- Инструмент получает собственный визуальный образ «Софит»: тёмная сцена, один
  мягкий луч света и крупная реплика в центре, с приятными анимациями и звуком.
- Инструмент получает отдельную страницу, карточку в каталоге и переход обратно
  в каталог.

## Capabilities

### New Capabilities

- `first-line-generator`: генерация открывающих реплик для сцены, тон-теги с
  фильтрацией, управление колодой без повторов и показ прогресса.

### Modified Capabilities

Нет.

## Impact

- Новая статическая страница и локальные данные в `tools/first-line/`.
- Новая карточка и графический идентификатор инструмента в корневом каталоге.
- Использование существующей общей навигации для возврата в каталог.
- Работа без фреймворка, сборки и сторонних зависимостей с относительными
  путями, совместимыми с `file://` и GitHub Pages.
