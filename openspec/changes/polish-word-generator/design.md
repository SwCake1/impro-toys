## Context

`tools/word/` — самостоятельная статическая страница без сборки и внешних зависимостей. Список существительных хранится в `app.js` строчными буквами, а интерфейс выводит слово и историю напрямую. Внизу страницы остаётся видимая отладочная строка.

## Goals / Non-Goals

**Goals:**

- Единообразно показать выбранные слова с заглавной буквы.
- Дать короткую звуковую обратную связь, похожую на бросок костей, при переходе между словами.
- Убрать отладочную подпись из пользовательского интерфейса.
- Сохранить работу страницы в современных настольных и мобильных браузерах, включая локальное открытие файла.

**Non-Goals:**

- Не менять состав, порядок или алгоритм перемешивания слов.
- Не добавлять внешние библиотеки, загрузку медиафайлов или настройки громкости.
- Не проигрывать звук при начальной загрузке или сбросе круга, когда новое слово не выбрано.

## Decisions

### Отделить представление от исходных данных

Слова в `WORDS`, колоде и истории останутся в исходном виде, а перед показом будут форматироваться отдельной функцией с русской локалью. Это сохранит текущую логику уникальности и возврата назад, но сделает регистр единым в основном поле и истории.

Альтернатива — переписать весь словарь с заглавной буквы. Она меняет данные без необходимости и повышает риск несогласованности при будущих дополнениях.

### Синтезировать звук средствами Web Audio API

После пользовательского действия перехода к следующему или предыдущему слову страница создаёт короткий приглушённый, затухающий шумовой эффект с несколькими лёгкими ударами. Он не требует отдельного звукового файла, сетевого доступа или изменений структуры `shared/` и будет запускаться в контексте разрешённого пользователем жеста.

Если Web Audio API отсутствует или браузер не разрешает воспроизведение, смена слова всё равно выполняется без сообщения об ошибке.

Альтернатива — добавить аудиофайл. Для одного короткого эффекта это добавит бинарный ресурс и потребует загрузки/управления воспроизведением без дополнительной пользы.

### Удалить отладочный элемент целиком

Элемент `#status`, его стили и запись в него из `init()` будут удалены. Техническое состояние не должно быть частью пользовательского интерфейса.

## Risks / Trade-offs

- [Некоторые браузеры ограничивают аудио до жеста пользователя] → звук вызывается только в обработчиках кнопок и клавиши пробела; выбор слова не зависит от успешного запуска аудио.
- [Синтезированный эффект может субъективно отличаться от настоящих костей] → использовать короткий низкогромкий шум и несколько затухающих ударов; не добавлять раздражающий длительный звук.
- [Капитализация может отличаться для нестандартных слов] → применить локальное преобразование только первой буквы, сохранив остальную строку как в словаре.
