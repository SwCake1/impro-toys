## Why

Для импровизационного рэп-баттла нужен быстрый способ получить понятную и
смешную тему с двумя противоборствующими сторонами. Новый инструмент позволит
ведущему одним действием запустить дебатный баттл без предварительной подготовки.

## What Changes

- Добавляется отдельное приложение «Рэп батл!» с генерацией случайной темы по
  кнопке.
- Добавляется локальная подборка минимум из 100 русскоязычных тем, каждая из
  которых сталкивает ровно две стороны.
- Тема отображается как две позиции с центральным `VS`; подряд не показывается
  одна и та же тема.
- Появление новой темы сопровождается локальным MP3-эффектом DJ-скретча без
  пустого хвоста и наложения при быстрых повторных нажатиях.
- Приложение получает самостоятельный визуальный стиль дерзкого андеграундного
  рэп-баттла.
- Инструмент добавляется в корневой каталог и сохраняет общий переход обратно к
  каталогу.
- Интерфейс работает без сети через `file://` и поддерживает настольные и
  мобильные браузеры, клавиатуру, указатель и касание.

## Capabilities

### New Capabilities

- `rap-battle-topic-generator`: Генерация и показ тем с двумя
  противоборствующими сторонами для импровизационного рэп-баттла.

### Modified Capabilities

Нет.

## Impact

- Новая папка инструмента в `tools/` со статическими HTML, CSS, JavaScript и
  локальными данными.
- Новая карточка и графический идентификатор в корневом каталоге.
- Использование существующего общего примитива навигации без новых зависимостей,
  сборщика, API или серверной части.
