## Why

Импровизаторы часто начинают сцену с заданной локации — места, где происходит действие. Сейчас в каталоге инструментов нет способа быстро получить случайную локацию: список из 104 локаций лежит в `tools/locations/locations.txt` как обычный текст и никак не используется. Нужен простой инструмент «одна кнопка → случайная локация», единообразно встроенный в существующий каталог.

## What Changes

- Добавляется новый инструмент **«Генератор локаций»** в `tools/locations/`: страница с одной крупной кнопкой, которая выдаёт случайную локацию из списка.
- Уникальное оформление в виде **split-flap табло** (перекидные буквы, как на вокзале/в аэропорту): при нажатии буквы «перекручиваются» до новой локации. Инструмент визуально отличается от остальных карточек каталога.
- Выдача **без повторов внутри круга**: перебор перемешанного списка, по исчерпании — новое перемешивание.
- Отображается **счётчик локаций** — текущая позиция в круге и общее число (например «17 из 104»).
- Данные локаций выносятся из `locations.txt` в `tools/locations/locations.js` в виде `window.LOCATIONS = [...]` (тот же приём, что `topics.js` в `rap-battle`), что гарантированно работает при открытии сайта через `file://` и на GitHub Pages.
- В каталог (`index.html`) добавляется запускаемая карточка инструмента с собственной иконкой.
- Поддержка указателя, касания и клавиатуры; относительные ссылки и ресурсы.

## Capabilities

### New Capabilities
- `location-generator`: инструмент, выдающий по запросу случайную локацию для начала сцены из встроенного списка, без повторов внутри круга, с оформлением в виде перекидного табло.

## Impact

- **Новые файлы**: `tools/locations/index.html`, `tools/locations/app.js`, `tools/locations/locations.js`, `tools/locations/styles.css` (при необходимости), иконка карточки в `shared/assets/`.
- **Изменяемые файлы**: корневой `index.html` (новая карточка каталога).
- **Данные**: `tools/locations/locations.txt` конвертируется в `tools/locations/locations.js`; исходный `.txt` может остаться как источник или быть удалён.
- **Зависимости**: отсутствуют — проект остаётся статическим сайтом без сборки и пакетного менеджера.
- **Совместимость**: работа через `file://` и на GitHub Pages; десктоп и мобильные браузеры.
